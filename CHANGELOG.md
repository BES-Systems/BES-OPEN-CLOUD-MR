# 0.2.0 - Feature Release: AI Tooling and Component Utilities

# 🚀 Features

*   **Component Explanation CLI:** Introduced a new command-line utility to generate detailed technical explanations for specific components.

*   Added comprehensive setup scripts and tooling to streamline local development environments.
*   Implemented structured commit messaging assistance and automated release generation pipelines.
